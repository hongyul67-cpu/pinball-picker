# 🎱 구슬 낙하 뽑기

발표 순서·당번·모둠을 재미있게 뽑는 추첨 도구입니다. 구슬이 못과 서로 부딪히며 떨어지는
물리 애니메이션으로 순위를 정합니다. 교실 화면에 띄워 쓰기 좋습니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/pinball-picker/

## 주요 기능
- **참가자 입력** — 이름 · 조(모둠) 이름
- **당첨 기준 / 제한 시간** 설정
- **장애물 · 물리** 옵션 조절
- **실시간 순위** 표시

## 사용 방법
- 위 주소를 열면 바로 사용할 수 있습니다. 설치가 필요 없습니다.
